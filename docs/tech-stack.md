# Technology Stack

## Frontend
- **Framework**: Next.js 14 (App Router)
- **State Management**: Zustand (Editor, Theme, Viewer)
- **Styling**: Tailwind CSS
- **3D Rendering**: Three.js, React Three Fiber (R3F), React Three Drei
- **Icons**: Lucide React
- **UI Components**: Custom Resizable Panels, Tabbed Sidebars

## Backend
- **Framework**: FastAPI (Python 3.10+)
- **Geometric Kernel**: OpenSCAD
- **AI Engine**: Google Gemini 3 (via `google-genai` SDK)
- **Environment**: `python-dotenv`
- **CORS**: Enabled for `http://localhost:3000`

## Infrastructure
- **Memory/Context**: Antigravity Workspace Memory (`.agent`, `.project`)
- **Version Control**: Git
