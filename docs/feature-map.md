# Feature Map

## 🏗️ Core Editor
- [x] **Viewport**: 3D scene with Three.js rendering.
- [x] **Gizmo**: Interactive translation/rotation controls.
- [x] **Scene Tree**: Visual hierarchy of parts.
- [x] **Properties Panel**: Interactive parameter editing for parametric parts.
- [x] **Undo/Redo**: Robust history engine.
- [x] **Keyboard Shortcuts**: Global hotkeys for productivity.
- [x] **Status Bar**: Real-time coordinate and scene feedback.

## 🤖 AI Assistant
- [x] **Chat Panel**: Natural language interface.
- [x] **Structured Commands**: AI-driven scene manipulation (Add, Transform, Duplicate, Delete).
- [x] **Transaction Logs**: Real-time feedback on AI actions.

## 📦 Geometry & Export
- [x] **Robotic Templates**: Chassis, Motor Mounts, Battery Trays.
- [x] **STL Import**: Support for static meshes.
- [x] **STL Export**: Multi-part assembly export via OpenSCAD.

## 🛠️ Workspace
- [x] **Resizable Layout**: Multi-dock system.
- [x] **Memory Management**: Persistence via `.agent/` and `.project/`.
